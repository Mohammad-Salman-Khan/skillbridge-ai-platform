# SkillBridge AI

## Theme
Sustainability & Social Impact

## Problem

Millions of students, especially from Tier-2 and Tier-3 cities, lack personalized career guidance. Existing platforms provide generic advice instead of personalized career roadmaps.

## Solution

SkillBridge AI is an AI-powered career mentor that analyzes a student's profile, interests, skills, and goals to generate:

- Personalized career roadmap
- Skill gap analysis
- Project recommendations
- Resume suggestions
- Internship recommendations

## Tech Stack

Frontend:
React + Tailwind CSS

Backend:
FastAPI

AI:
Google Gemini API

Database:
